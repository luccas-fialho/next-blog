# NextJS Blog

NextJs Blog created for study purposes, It was a greate way to practive my language skills. It was used NextJS, Actions and Mockup data with SQLite db.

It has Actions for all CRUD cycle and a login system.

## Techs used

- Next 15
- React 19
- React-dom 19
- TailwindV4
- Drizzle ORM
- Slugify
- Bcrypt

## How to run the blog

First you have to have Node 18+ installed.

```bash
cp .env.local-example .env.local

# change the env variables for yours

npm ci

npm run migrate

npm run seed

npm run dev
```
